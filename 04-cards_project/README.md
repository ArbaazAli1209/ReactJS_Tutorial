# Cards Project

A simple React + Vite project that showcases a collection of job cards for different companies. The app demonstrates how to render repeated UI elements from data, pass props between components, and build a clean reusable card layout.

## Demo

- Live demo: https://cardsproject-seven.vercel.app/

## Features

- Responsive job card layout
- Reusable card component
- Company branding, role details, tags, salary, and location information
- Clean UI built with React and custom styling

## Tech Stack

- React
- Vite
- Lucide React
- ESLint

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the local URL shown in the terminal in your browser.

## Available Scripts

- `npm run dev` - Start the Vite development server
- `npm run build` - Create a production build
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint checks

## Project Structure

- `src/App.jsx` - Contains the job data and renders the list of cards
- `src/components/Card.jsx` - Reusable card component used for each listing
- `src/index.css` - Global styling for the app

## Purpose

This project is part of a React learning journey and focuses on component-based UI development, props, and list rendering.
